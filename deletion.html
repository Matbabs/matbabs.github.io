Suppression de votre compte

1. Suppression par e-mail

Si vous souhaitez supprimer votre compte A BLOC et toutes les données associées, vous pouvez le faire directement dans les paramètres de l'application. Si vous n'avez plus accès à l'application, vous pouvez nous envoyer votre demande par e-mail à : matissebabonneau@gmail.com. Nous supprimerons votre compte et vos données sous 48h.

2. Suppression via l'application (Recommandé)

1. Rendez-vous sur le profil

2. Sélectionnez supprimer le compte

3. Confirmez SUPPRIMER DÉFINITIVEMENT

Êtes-vous sûr de vouloir supprimer votre compte ? Cette action est irréversible
